📘 How to Use

Step 1 — Customize Your Crosshair
Use the sliders on the main page to adjust size, shape, RGB speed, and glow. You can preview changes in real-time before generating the script.
Step 2 — Generate the Script
Click the Generate Script button to create your custom crosshair code. This code is tailored based on your selected settings.
Step 3 — Copy the Code
Click Copy Code to instantly copy the generated script to your clipboard.
Step 4 — Open Tampermonkey
Launch the Tampermonkey extension in your browser. This tool allows you to run custom scripts on websites.
Step 5 — Create New Script
Click Create a new script inside Tampermonkey and remove any default code.
Step 6 — Paste & Save
Paste your copied code and press Ctrl + S to save the script.
Step 7 — Open Your Game
Visit a supported browser game (like surviv.io or similar).
Step 8 — Reload the Page
Refresh the page if needed — your custom RGB crosshair will now appear in-game.
⚡ Tip: Make sure Tampermonkey has "Run Userscripts" perms.
